## Installation

```shell
cd ~/EasonDrone/Modules
git clone https://github.com/HuaYuXiao/EasonDrone_Msgs.git
cd ~/EasonDrone
catkin_make install --source Modules/EasonDrone_Msgs --build Modules/EasonDrone_Msgs/build
```
